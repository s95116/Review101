Common Git Commands

1) git clone -- clones the repository from a url provided from git.com(beginning stages)

2) git add. -- adds all files currently in the folder to the git repository. You can also specify a specifc file by typing git add + filename.

3) git commit -m"git commit message goes in qutoes" -- notes a change to the local repository. Usually the last step before pushing the commit to the repository online.

4) git push or git push origin master -- pushes the changes to the git repository(hosting service).

5) git pull -- pulls the latest version from the repository so you have the latest code on your workspace.
